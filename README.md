# 🎓 StudentPortal

A simple **ASP.NET Core MVC** CRUD application to manage student information.

---

## 🚀 Technologies Used

- 💻 **C#**
- 🌐 **ASP.NET Core MVC**
- 🎨 **HTML, CSS**
- 🗄 **Microsoft SQL Server**
- 🔗 **Entity Framework Core**

---

## 📚 Features

- ✅ Add new student
- ✅ View student list
- ✅ Update student information
- ✅ Delete student record
- ✅ Server-side form validation
